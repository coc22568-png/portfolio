# 财报分析指南

深度解读财报，发现数字背后的故事与信号。

---

## 场景判断（第一步必须执行）

| 用户输入 | 场景 | 动作 |
|---------|------|------|
| 股票代码/公司名 + "财报"/"季报"/"年报" | 场景S：股票财报 | 读取 references/stock-report.md |
| 基金代码/名称 + "报告"/"季报"/"年报" | 场景F：基金报告 | 读取 references/fund-report.md |
| 财报PDF文件 | 场景D：文档解读 | 提取内容后按股票/基金分类 |

---

## 核心原则

1. **读数字，更读变化**：同比、环比、与预期比
2. **读结果，更读原因**：什么驱动了业绩？
3. **读过去，更读未来**：guidance、管理层表述、战略方向
4. **验证一致性**：管理层说的和做的是否一致

---

## 与股票/基金分析指南的分工

| 场景 | 使用Skill | 说明 |
|------|----------|------|
| "分析一下腾讯" | 股票分析指南 | 完整基本面分析（估值、护城河等） |
| "解读腾讯最新季报" | 财报分析指南 | 专注单份财报的深度解读 |
| "这只基金怎么样" | 基金投资指南 | 基金产品选择分析 |
| "看看这只基金的季报" | 财报分析指南 | 专注基金报告解读 |

财报分析指南是**工具型Skill**，为股票/基金分析提供数据支持。

## 场景T：科技股财报深度分析

用户请求「科技股深度分析」「NVDA/META/MSFT等财报」「16模块分析」「多视角投资备忘录」「Key Forces识别」时，读取 references/tech-earnings.md 执行完整分析流程。

与普通财报解读的区别：
- 普通财报（场景S）：结构化解读一份财报的关键数据
- 科技股深度（场景T）：Key Forces驱动 + 16模块 + 6大投资哲学视角 + 估值矩阵 + 可执行决策框架

适用标的：美股科技公司（NVDA、META、MSFT、AAPL、GOOGL、AMZN、TSLA等）及其他需要机构级深度尽调的标的。
