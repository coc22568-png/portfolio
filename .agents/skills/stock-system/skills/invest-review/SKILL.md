# invest-review：复盘系统

> 亏损不可怕，不知道为什么亏才可怕。复盘的目的不是自我批判，是找到可以改进的具体一件事。

---

## 复盘原则

1. **对事不对人**：复盘交易行为，不评判自我
2. **聚焦可控因素**：运气无法复盘，决策过程才能改进
3. **一次只改一件事**：找到最重要的那个问题，不贪多
4. **记录优于记忆**：复盘要落文字，否则三个月后你会忘记

**问题1：当初为什么买？**

**问题2：结果发生了什么？**

**问题3：哪里做对了？**

```

**问题4：最关键的一个错误是什么？**

```

## 模块二：错误类型分类库

将每次复盘的错误归类，长期积累形成自己的"错误地图"：

### 2.1 认知层错误

| 错误类型 | 症状 | 改进方向 |
|---------|------|---------|
| 过度自信 | 仓位过重、没有止损 | 降低单股仓位上限，强制设止损 |
| 确认偏误 | 只看支持买入的信息 | 强制列出3条反对理由再做决定 |
| 高估能力圈 | 买了不懂的行业/公司 | 三关第一关更严格执行 |
| 忽视宏观 | 好公司买在熊市顶部 | 先调用 invest-meta 再选股 |
| 估值锚定错误 | 用行情高点PE作为基准 | 严格用历史分位而非绝对数值 |

### 2.2 情绪层错误

| 错误类型 | 症状 | 改进方向 |
|---------|------|---------|
| 追涨杀跌 | 涨了才买，跌了才卖 | 写好买卖计划再下单，不看盘下单 |
| 损失厌恶 | 止损执行不了 | 止损条件预先设好，设置提醒 |
| 踏空焦虑 | 没买到的股票越涨越后悔 | 记录"没有参与的机会"，分析是否该参与 |
| 沉没成本 | 亏了不愿割 | 每月问自己"如果今天是新股，我还会买吗？" |
| 短视近利 | 频繁交易，交了太多手续费 | 统计年度手续费，感受实际成本 |

### 2.3 执行层错误

| 错误类型 | 症状 | 改进方向 |
|---------|------|---------|
| 计划不执行 | 制定了计划却按情绪操作 | 交易前写下计划，事后对账 |
| 仓位失控 | 单股超过15% | 硬性设上限，超过自动减仓 |
| 分散不足 | 全押一两只股票 | 强制3只以上才满仓 |
| 分散过度 | 持有20只以上，不知道在买什么 | 持仓上限10只，精选重仓 |

## 模块四：错失机会分析

当你"看到了但没买"的股票大涨时，不要懊恼，要复盘：

```
股票：____    错失涨幅：____%

当时为什么没买？
  □ 没看到这个机会
  □ 看到了但分析结论是不买（理由：____）
  □ 看到了分析也对，但没执行（情绪：____）

如果是"分析结论是不买"：
  → 复盘当时分析哪里有盲区？
  → 下次遇到类似机会，分析框架需要补充什么？

如果是"没执行"：
  → 当时的情绪状态是什么？
  → 执行障碍是什么？如何降低执行摩擦？

结论：这是可复制的机会吗？下次如何捕捉？____
```

## 复盘知识回流协议

复盘发现的重要认知，应回流到对应技能：

```
发现宏观判断有盲区 → 更新 invest-meta 的分析维度
发现某类股票总选错 → 更新 invest-scout 的过滤条件
发现某类公司分析错 → 更新 invest-stock 的三关标准
发现执行总出问题  → 更新 invest-trader 的操作规则
```
