# invest-trader：交易决策与风控

> 分析做得再好，执行上一塌糊涂也白费。买在哪、卖在哪、仓多重，这三个问题决定最终收益。

---

## 前置条件

本技能**必须依赖** `invest-stock` 的分析结论才能运作：
- 股票是否通过三关审查？
- 合理估值区间是多少（PE/PB历史分位参考）？
- 当前价格相对合理估值的位置？

无分析结论 → 拒绝输出操作建议，先调用 invest-stock。

## 模块一：仓位管理

### 1.1 总仓位决策（基于市场温度）

| 市场环境 | PE历史分位 | 建议总仓位上限 |
|---------|-----------|-------------|
| 极度低估 | <20% | 70-100%（可满仓）|
| 低估 | 20-40% | 50-70% |
| 合理 | 40-60% | 30-50% |
| 高估 | 60-80% | 10-30% |
| 极度高估 | >80% | 0-10%（空仓为主）|

**注意**：总仓位是上限，不是目标。个股质量不够不要强行填满上限。

### 1.2 单股仓位计算（改良凯利公式）

```
原始凯利公式：f = (p × b - q) / b
  p = 胜率（估计）
  q = 1 - p（败率）
  b = 盈亏比（止盈幅度 / 止损幅度）

实用简化版（保守型）：
  凯利仓位 × 0.25 = 建议单股仓位上限

示例：
  胜率 60%，目标收益 30%，止损 10%
  凯利 f = (0.6 × 3 - 0.4) / 3 = 46.7%
  保守调整后 = 46.7% × 0.25 ≈ 11.7%
  → 单股仓位不超过 12%
```

### 1.3 单股仓位上限规则（硬性限制）

```
个股集中度限制（不可突破）：
  □ 单只股票 ≤ 15%（防单一风险）
  □ 同一行业 ≤ 30%（防行业风险）
  □ 强相关性的两只股票合计 ≤ 20%

仓位分级：
  重仓（10-15%）：通过三关审查 + 安全边际充足 + 高景气行业
  标配（5-10%）：通过三关审查 + 估值合理
  轻仓（2-5%）：三关审查有疑虑 / 用于跟踪观察
  试仓（1-2%）：新标的，先建小仓感受流动性
```

## 模块二：买入策略

### 2.1 买入时机三个条件（需同时满足）

```
条件1：基本面确认（来自 invest-stock）
  ✅ 通过三关审查
  ✅ 当前PE/PB处于历史合理区间（< 60%分位）

条件2：技术面辅助确认
  ✅ 不追高（股价未偏离20日均线超过15%）
  ✅ 成交量未出现极端放量（未超过30日均量的3倍）
  ✅ 大盘情绪不处于极度亢奋（新高次日不追）

条件3：催化剂或安全边际
  ✅ 有明确的价值回归路径（业绩增长 / 估值修复 / 事件催化）
  OR
  ✅ 当前股息率 > 3.5%（股息提供安全边际）
```

### 2.2 建仓方式

**方式A：分批建仓（推荐）**
```
第一批（试仓）：目标仓位的 25%
  时机：确认买入信号后立即买入
  
第二批（加仓）：目标仓位的 50%
  时机：价格下跌 5-8% OR 出现明确催化剂

第三批（满仓）：目标仓位的 25%
  时机：价格进一步下跌 OR 业绩验证

原则：越跌越买（前提：逻辑未变），不在上涨中加仓
```

**方式B：一次性建仓（仅适用以下情形）**
```
适用条件（全部满足）：
  □ 估值处于历史极低分位（< 20%）
  □ 安全边际充足（股息 > 4% 或 PE < 10倍）
  □ 仓位较小（ < 5%）
  □ 市场整体处于低估区域
```

### 2.3 不买清单（触发任一项 → 不操作）

```
❌ 股价近5日涨幅 > 20%（追高风险）
❌ 大盘当日跌幅 > 2%（系统性风险上升时不新建仓）
❌ 刚公布重大负面消息（等待情绪稳定）
❌ 距离财报披露 < 2周（避免信息不对称）
❌ 近期有重大解禁（大股东减持压力）
```

## 模块三：止盈止损

### 3.1 止损设置

```
主动止损规则（硬性执行，不可妥协）：

类型A：价格止损
  单股亏损 > 15% → 减仓至一半
  单股亏损 > 25% → 清仓
  
类型B：逻辑止损（比价格止损更重要）
  触发任一项立刻止损：
  □ 原来买入的逻辑已经不成立
  □ 实控人/大股东大额减持（连续两次超过1%）
  □ 发现财务造假线索
  □ 核心业务突然恶化（营收连续两季度 > 20%下滑）
  □ 行业遭遇强监管整顿

类型C：时间止损
  持有超过12个月，股价未有明显变化，且逻辑未兑现
  → 重新评估，考虑换仓到更有确定性的标的
```

### 3.2 止盈策略

```
策略A：目标价止盈
  预设合理价值（基于PE估值中枢），达到目标价后：
  - 达到目标价 → 减仓 50%
  - 超出目标价 15% → 再减仓 25%
  - 超出目标价 30% → 清仓剩余

策略B：动态止盈（趋势跟踪型）
  适用于景气度持续上行的周期/成长股：
  - 设置追踪止盈：股价从高点回撤 > 20% → 减仓
  - 保留一部分仓位享受超额涨幅

策略C：分红替代止盈（高股息股）
  以股息回收成本：
  - 持有年数 × 股息率 ≥ 买入成本的 40% → 可持续持有
  - 当股息率降至 < 2%（股价涨太多）→ 开始减仓
```

## 模块四：资金配置计划

### 4.1 总资产配置框架

```
输入：用户总可投资资金 = X 万元
      当前市场温度（来自 invest-meta）
      候选标的数量和质量（来自 invest-scout + invest-stock）

配置步骤：
  Step 1: 确定股票仓位上限（依据市场温度，见1.1）
  Step 2: 预留应急资金（建议 ≥ 3个月生活费不投股市）
  Step 3: 按单股仓位限制分配
  Step 4: 保留一定"子弹"（建议保留20%现金等待机会）
```

### 4.2 组合构建示例

```
假设：可投资100万，市场处于低估区间，有3只通过审查的股票

总仓位上限：70%（低估区间标准）= 70万
  A股（重仓，高景气行业龙头）：15万（15%）
  B股（标配，估值合理）：10万（10%）
  C股（标配，高股息价值）：10万（10%）
  预留股票仓位空间（等待更好机会）：35万
  现金缓冲：30万（30%）
```

### 4.3 再平衡规则

```
触发再平衡的条件（任一满足）：
  □ 某只股票涨跌幅导致仓位偏离目标 > 50%
     （目标10% → 实际 > 15% 或 < 5%）
  □ 市场温度发生档位变化（低估→合理，或合理→高估）
  □ 基本面发生重大变化（季报超预期/不及预期）
  □ 季度末例行检视
```
