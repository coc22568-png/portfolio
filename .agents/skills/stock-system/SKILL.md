# 股票投资全框架系统

> 投资是认知变现。认知=宏观判断×行业理解×公司分析×执行纪律。

---

## 四层架构

```
┌────────────────────────────────────────────────┐
│  invest-meta（元认知层）                         │
│  宏观·政策·事件·市场情绪·行业轮动知识库            │
└─────────────────────┬──────────────────────────┘
                      │ 输出：市场环境结论 + 行业优先级
┌─────────────────────▼──────────────────────────┐
│  invest-scout（选股层）                          │
│  宏观→行业→个股漏斗筛选，生成候选池               │
└─────────────────────┬──────────────────────────┘
                      │ 输出：候选股票列表 + 关注理由
┌─────────────────────▼──────────────────────────┐
│  invest-stock（分析层，已有）                     │
│  单股三关审查：懂不懂·好不好·贵不贵               │
└─────────────────────┬──────────────────────────┘
                      │ 输出：买/不买/观望 + 目标价区间
┌─────────────────────▼──────────────────────────┐
│  invest-trader（执行层）                         │
│  仓位配置·买入计划·止损止盈·风险控制              │
└─────────────────────┬──────────────────────────┘
                      │ 输出：具体操作指令
┌─────────────────────▼──────────────────────────┐
│  invest-review（复盘层）                         │
│  交易复盘·策略迭代·认知更新·错误归档              │
└────────────────────────────────────────────────┘
```

## 技能路由

| 用户说 | 路由到 |
|--------|--------|
| 「宏观怎么样」「政策影响」「市场环境」 | invest-meta |
| 「找股票」「哪些板块有机会」「选股」 | invest-scout |
| 「分析这只股票」「XX怎么样」 | invest-stock（已有）|
| 「怎么买」「买多少」「止损怎么设」「仓位」 | invest-trader |
| 「复盘」「回顾」「这次操作对不对」 | invest-review |

---

## 核心原则

1. **元认知先行**：每次分析前先确认当前市场环境（牛/熊/震荡）
2. **自上而下**：宏观→行业→个股，不做脱离环境的孤立分析
3. **决策留给用户**：分析框架和风险揭示是责任，替用户做决定是越界
4. **复盘闭环**：每笔交易都要复盘，认知才能迭代
