# invest-reit：REITs分析

> REITs是"不动产的股票化"。看的是底层资产质量，吃的是租金现金流。

---

## 时间定位（动态算法）

当前日期 T，自动推导数据基准：

```
财年锚点 N = T.year if T > 4/30 else T.year - 1
分析周期 = [N-4, N-3, N-2, N-1, N]
```

**REITs特殊说明**：
- 国内公募REITs2021年才启动，历史数据有限
- 季报披露时间为季度结束后1个月内
- 分派率按年化计算

---

## 三关审查

### 第一关：懂不懂（底层资产）

**第一性原理**：不懂底层资产，就不知道租金从哪来。

**验证问题**：
- 底层资产是什么？（高速公路/产业园/仓储物流/保障房/新能源）
- 资产在哪里？（一二线城市/三四线/偏远地区）
- 租金收入结构？（单一租户依赖度？租约期限？）

**资产类型检查**：

| 资产类型 | 关键指标 | 风险点 |
|---------|---------|--------|
| 高速公路 | 车流量、收费标准、剩余收费年限 | 疫情冲击、新能源车普及影响 |
| 产业园 | 出租率、租金水平、 tenant 质量 | 经济周期、产业外迁 |
| 仓储物流 | 出租率、租金、区域位置 | 电商增速放缓、供给过剩 |
| 保障房 | 租金稳定、政府补贴 | 租金管制、流动性差 |
| 新能源 | 发电小时数、电价、补贴政策 | 补贴退坡、技术迭代 |

**关键数据**：
- 底层资产个数：___个
- 主要资产占比：___%（前三大）
- 加权剩余租约：___年
- 前五大 tenant 占比：___%（集中度风险）

**未通过**：资产不透明或过于集中 → 停止分析，不买

---

### 第二关：好不好（运营与财务）

**第一性原理**：REITs的价值来自稳定现金流。

**运营指标**：

| 指标 | 当前 | 历史均值 | 判断标准 |
|------|------|---------|---------|
| 出租率 | ___% | ___% | > 90%优秀，< 80%危险 |
| 租金收缴率 | ___% | ___% | > 95%健康 |
| 租金增长率 | ___% | ___% | 与通胀挂钩或稳定增长 |
| 分派率 | ___% | ___% | > 4%有吸引力，> 6%需警惕风险 |

**财务指标**：

| 指标 | 当前 | 判断 |
|------|------|------|
| 资产负债率 | ___% | < 40%安全，> 50%警惕 |
| 利息覆盖倍数 | ___倍 | > 3倍安全 |
| 可供分配金额/净利润 | ___% | > 90%（REITs要求） |
| 现金流稳定性 | 过去8季度波动 | 标准差< 15%稳定 |

**一票否决项**：
- 出租率连续下滑且低于80%
- 分派率> 8%但不可持续（处置资产一次性收益）
- 资产负债率> 50%且利息覆盖< 2倍
- 存在重大法律纠纷或资产减值风险

**未通过**：运营恶化或财务风险 → 停止分析，不买

---

### 第三关：贵不贵（估值与折价）

**第一性原理**：REITs估值看分派率与净资产。

**估值指标**：

| 指标 | 当前 | 发行时 | 判断 |
|------|------|--------|------|
| 市场价格 | ___元 | ___元 | 溢价/折价___% |
| NAV（净资产价值） | ___元 | - | 市净率___倍 |
| 分派率 | ___% | ___% | 与发行时比较 |
| 与十年期国债利差 | ___% | - | > 3%有吸引力 |

**估值方法**：

**分派率法**：
- 合理分派率 = 无风险利率 + 风险溢价（通常3-4%）
- 当前分派率___% vs 合理分派率___%

**NAV法**：
- 若市价 < NAV（折价），可能有机会
- 若市价 > NAV 20%以上（高溢价），需警惕

**同类比较**：

| REIT名称 | 资产类型 | 分派率 | 溢价率 | 出租率 |
|---------|---------|--------|--------|--------|
| 本基金 | ___ | ___% | ___% | ___% |
| 同类平均 | - | ___% | ___% | ___% |

**未通过**：高溢价（> 30%）+ 低分派率（< 4%） → 等待或放弃

## 盲区与局限

1. 国内公募REITs历史短，缺乏完整周期验证
2. 流动性较差，大额交易可能冲击价格
3. 扩募机制尚不完善，成长性有限
4. 税收政策变化可能影响分派率

## 核心原则

1. **底层资产第一**： location, location, location
2. **现金流稳定**：出租率和租金增长是核心
3. **分派率可持续**：区分经常性收益和一次性收益
4. **折价买入**：高溢价时风险大于机会
5. **流动性注意**：REITs流动性不如股票，大额交易需谨慎
