# invest-meta：元认知引擎

> 投资的第一步不是选股，是判断当前站在什么地方。环境判断错，个股分析再准也是白费。

## 模块一：市场环境定位

### 1.1 经济周期定位（美林时钟简化版）

| 象限 | 特征 | 领先资产 | 滞后资产 |
|------|------|---------|---------|
| 复苏期 | GDP↑ 通胀低 | 股票（周期/成长）| 债券 |
| 过热期 | GDP↑ 通胀↑ | 大宗商品 | 债券/成长股 |
| 滞胀期 | GDP↓ 通胀↑ | 现金/防御股 | 成长股/债券 |
| 衰退期 | GDP↓ 通胀低 | 债券/消费防御 | 周期股 |

**判断方法**：
```
Step 1: GDP增速趋势（加速/减速？）→ 搜索"最新GDP数据/PMI"
Step 2: 通胀趋势（CPI/PPI方向）→ 搜索"最新CPI PPI数据"
Step 3: 信用周期（社融/M2扩张还是收缩？）→ 搜索"最新社融数据"
Step 4: 综合定位象限
```

### 1.2 A股市场情绪定位

**估值温度计**（以沪深300为基准）：

| 区间 | PE分位 | 信号 | 操作方向 |
|------|--------|------|---------|
| 极度低估 | <20% | 强烈买入信号 | 重仓 |
| 低估 | 20-40% | 偏多 | 标配+加仓 |
| 合理 | 40-60% | 中性 | 标配 |
| 高估 | 60-80% | 偏谨慎 | 减仓 |
| 极度高估 | >80% | 强烈减仓信号 | 轻仓/空仓 |

**情绪指标补充**：
- 换手率（高换手=情绪过热）
- 新开户数（历史高位=顶部信号）
- 融资余额变化（加杠杆速度）
- 北向资金连续流向

## 模块二：政策解读框架

### 2.1 政策信号分类

```
一级信号（最高权重）：
  - 中央政治局会议定调
  - 政府工作报告关键词
  - 央行货币政策委员会表述

二级信号（重要参考）：
  - 国务院专项政策文件
  - 发改委、工信部、证监会重要通知
  - 财政部预算和税收调整

三级信号（行业催化）：
  - 部委专项支持政策
  - 地方政府实施细则
  - 行业协会规范文件
```

### 2.2 政策解读四问

1. **受益方向**：这个政策明确支持/限制哪个方向？
2. **力度强弱**：是"研究"、"鼓励"还是"强制"、"专项资金"？
3. **时间节奏**：政策从发布到落地通常需要多久？是否已被市场price-in？
4. **传导路径**：政策→行业供需→上下游企业→利润变化的逻辑链条是否清晰？

## 模块三：事件驱动评估

### 3.1 事件分类与影响时窗

| 事件类型 | 案例 | 市场影响时窗 | 处理方式 |
|---------|------|------------|---------|
| 突发黑天鹅 | 地缘冲突、疫情 | 短期剧烈→逐步消化 | 等待情绪稳定后评估 |
| 政策转向 | 货币宽松/收紧 | 中长期趋势性 | 重新定位经济周期 |
| 行业监管 | 反垄断、教培整顿 | 中期持续压制 | 退出被监管行业 |
| 技术革命 | AI突破、新能源爆发 | 长期结构性 | 寻找受益标的 |
| 季节性事件 | 财报季、年末效应 | 短期规律性 | 把握节奏 |

## 模块四：行业轮动追踪

### 4.1 当前行业景气度扫描

搜索以下维度评估各行业：
- **景气度**：行业PMI、产销数据、价格走势
- **政策支持度**：近3个月政策提及频次
- **资金流向**：北向/机构近期净流入前5行业
- **估值分位**：各行业PE/PB历史分位

### 4.2 行业轮动规律（经济周期驱动）

```
复苏初期  → 金融、地产（信用扩张先行）
复苏中期  → 消费、医疗（需求恢复）
过热期    → 资源、能源、周期（通胀受益）
衰退初期  → 公用事业、消费防御（避险）
衰退深期  → 债券、现金（保守）
政策刺激  → 基建、新能源、半导体（政策催化）
```
