---

## 使用说明

1. 每位持仓或重点观察经理建一个档案
2. 每次季报发布后更新「年检记录」
3. 「卖出触发条件」在买入时就写好，不要等到需要卖的时候再想
4. 档案存放路径由用户自定义（建议与投资笔记放在一起）
