# invest-scout：选股雷达

> 选股不是在茫茫股海随机捞鱼，而是先确定鱼在哪片水域，再下网精准捕捞。

---

## 前置条件

**先调用 invest-meta 确认市场环境**，获取：
- 当前经济周期象限
- 政策重点支持方向
- 近期重大事件影响板块
- 市场估值温度

若用户已提供环境背景，可跳过此步骤直接进入漏斗。

---

## 选股漏斗（自上而下三层过滤）

```
第一层：宏观/政策/事件过滤
  ↓ 确定当前受益的大方向（2-3个）
第二层：行业精选
  ↓ 在大方向中找景气度高、格局好的细分赛道
第三层：个股候选
  ↓ 在细分赛道中找财务质量好、估值合理的公司
```

---

## 第一层：确定大方向

### 1.1 四类机会来源

**A. 周期受益（宏观驱动）**
```
经济复苏 → 关注：金融、地产链、消费、周期资源
流动性宽松 → 关注：成长股、科技、高负债修复
出口景气 → 关注：外向型制造业、电子、纺织服装
```

**B. 政策催化（政策驱动）**
```
筛选标准：
  □ 政策级别是否为一级（中央政治局/国务院）
  □ 政策方向是否明确（强制性/专项资金）
  □ 是否有可量化的目标（如"2030年达到XX"）
  □ 产业链受益是否清晰
```

**C. 事件驱动（催化剂驱动）**
```
积极事件：行业并购整合、技术突破、海外拓展、大客户公告
消极事件规避：监管趋严、核心竞争力消失、行业供给过剩
```

**D. 价值重估（估值驱动）**
```
条件：基本面稳定 + 估值历史低分位 + 股息率较高
典型：红利策略、低PB国企、困境反转
```

---

## 第二层：行业精选

### 2.1 行业景气度评分表

搜索以下数据，对候选行业打分（每项1-5分，满分25分）：

| 评分维度 | 权重 | 满分 | 数据来源 |
|---------|------|------|---------|
| 行业景气度（PMI/产销/价格） | 高 | 5分 | 行业数据/月报 |
| 政策支持力度 | 高 | 5分 | 政府文件频次 |
| 供需格局（竞争格局是否改善） | 高 | 5分 | 行业研究 |
| 估值历史分位（越低越好） | 中 | 5分 | 行业PE历史 |
| 资金流入趋势（北向/机构） | 中 | 5分 | 资金流向数据 |

**15分以上**：重点关注
**10-15分**：一般关注
**10分以下**：暂不关注

### 2.2 行业格局判断（波特五力简化）

```
一句话判断：这个行业「谁说了算」？

□ 上游原材料卡脖子？（矿产、芯片等）
□ 下游客户集中度高？（大客户依赖）
□ 行业内是否存在明显龙头？（集中度提升趋势）
□ 新进入者壁垒高低？（技术/资金/牌照门槛）

理想行业：格局稳定 + 龙头强者恒强 + 新进入者难以复制
```

## 第三层：个股候选

### 3.1 初步财务过滤器（快速排除）

以下任一条满足 → **直接排除，不进入候选池**：

```
❌ 过去3年有2年以上亏损
❌ 资产负债率 > 70%（非金融企业）
❌ 应收账款 / 净利润 > 300%
❌ 实控人近1年内大额减持（累计 > 5%）
❌ 存在重大违规/处罚记录
❌ 审计意见非标（非标准无保留意见）
```

### 3.2 候选股正向筛选条件

以下条件满足越多越好（满分6项）：

```
✅ ROE连续3年 > 12%
✅ 毛利率 > 25% 且稳定/提升
✅ 经营现金流/净利润 > 80%
✅ 资产负债率 < 40%
✅ 所在行业赛道景气分 > 15分
✅ 当前PE/PB处于历史低分位（< 40%）
```

满足4项以上 → 进入候选池，提交给 `invest-stock` 深度分析

### 3.3 行业龙头识别

```
龙头判断依据：
  1. 市占率（行业前3，最好第1）
  2. 品牌溢价（同类产品能否卖更贵）
  3. 客户粘性（客户切换成本高低）
  4. 规模效应（收入增长→成本下降→利润弹性大）
  5. 研发投入（能否持续维护技术壁垒）
```

## 特殊选股策略

### 策略A：困境反转筛选

```
目标：找出基本面正在改善但市场尚未意识到的公司

条件：
  □ 过去2年业绩不佳（受行业/周期压制，非商业模式问题）
  □ 最新季度有改善迹象（营收/利润环比转正）
  □ 行业景气度开始回升
  □ 估值处于历史极低分位（PE/PB < 20%）
  □ 大股东近期增持或公司回购
```

### 策略B：高股息价值筛选

```
目标：以股息率为主要安全边际，适合震荡/熊市环境

条件：
  □ 股息率 > 3.5%（高于10年期国债收益率）
  □ 过去5年持续分红（分红比例 > 40%）
  □ 自由现金流 > 净利润（分红有保障）
  □ 负债率低（< 40%，保证分红能持续）
  □ 国企/央企背景（分红稳定性更高）
```

### 策略C：事件驱动快速捕捉

```
目标：在重大事件后3-7个交易日内找到合理标的

步骤：
  1. 明确事件性质（政策/技术/并购/海外冲击）
  2. 梳理产业链受益逻辑（直接受益 vs 间接受益）
  3. 直接受益方优先（逻辑最清晰，弹性最大）
  4. 过滤财务质量差的公司（事件驱动不改变烂公司命运）
  5. 评估市场是否已充分定价（避免追高）

注意：事件驱动是催化剂，不是基本面。持有周期通常短于基本面投资。
```
